# 🚆 Train Seat Booking System (Python)

A simple command-line based Train Seat Booking and Status Checking System built using Python. This project simulates basic functionalities of a railway reservation system.

## 📌 Features
- ✅ Check available seats for a train
- ✅ Book tickets
- ✅ Handles invalid train numbers
- ✅ Prevents overbooking
- ✅ User confirmation before booking

## 🚀 How It Works
- User enters a train number
- System checks seat availability
- If valid, user is asked whether they want to book tickets
- If yes → booking is processed
- If no → program exits gracefully
